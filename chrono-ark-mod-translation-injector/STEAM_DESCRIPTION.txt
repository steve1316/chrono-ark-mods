[h1]Mod Translation Injector[/h1]

Injects English translations for mod content in Chrono Ark at runtime. Supports both localization key overrides and direct hardcoded CJK string replacement.

This mod is the successor to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3427597117]Alternative English Translations[/url] ([url=https://github.com/steve1316/chrono-ark-alternative-english-translations]GitHub[/url]), which has been deprecated. The old approach relied on manually patching mod files with fragile scripts that broke whenever mods updated. This injector instead patches translations at runtime — no mod files are modified, and updates to mods don't break anything.

[h2]Features[/h2]

[olist]
[*] [b]Keyed Localization Overrides[/b] — Reads [code]keyed_overrides.json[/code] to replace mod localization keys with English text. Handles skill names, descriptions, character names, passive descriptions, and more.
[*] [b]Hardcoded Text Replacement[/b] — Reads [code]text_overrides.json[/code] to replace hardcoded CJK strings in UI text components with English translations. Covers TextMeshPro, legacy Text, and skill selection prompts.
[/olist]

[h2]Setup[/h2]

Place your translation JSON files in the mod's folder alongside the DLL.

[h3]keyed_overrides.json[/h3]

Localization key overrides grouped by mod name:

[code]
{
  "SomeMod": {
    "Skill_FireBall_Name": "Fireball",
    "Skill_FireBall_Desc": "Deals fire damage to a single enemy."
  },
  "AnotherMod": {
    "Item_Potion_Name": "Healing Potion"
  }
}
[/code]

[h3]text_overrides.json[/h3]

Hardcoded CJK string replacements grouped by mod name:

[code]
{
  "SomeMod": {
    "\u706b\u7403": "Fireball",
    "\u653b\u51fb": "Attack"
  }
}
[/code]

The mod name keys are for organization only — all entries are flattened into a single lookup at load time.

[h2]How Translations Are Made[/h2]

Translations are generated using [b]Claude Sonnet 4.6[/b] via the [url=https://github.com/steve1316/chrono-ark-translator]Chrono Ark Translator[/url], a full-stack translation management tool built specifically for localizing Chrono Ark Workshop mods.

The translator extracts strings from mod CSVs, DLLs, and GData JSON files, then processes them through AI with a two-tier glossary system — a global base-game glossary for consistent Chrono Ark terminology and per-mod custom glossaries for character-specific terms. Translations are processed in batches with glossary review between each batch to catch and correct terminology.

Not all mods are purely machine-translated. Many have been manually reviewed and redone with reference to their source material (wiki pages, etc.) to ensure ability names and terms are accurate.

In addition, some mods were good enough that only untranslated strings from their GData JSON files and DLL file were translated and the CSVs were kept in-tact.
